📁 Guia Operacional – Repositório de Provas

🎯 OBJETIVO

Este repositório serve como armazenamento organizado e permanente de provas (imagens, vídeos e documentos) vinculadas à denúncia ambiental.

O acesso principal sempre será feito através do PDF protocolado.

📌 PADRÃO DE NOMES (OBRIGATÓRIO)

Utilizar sempre:

  • letras minúsculas
  • sem acentos
  • sem espaços
  • usar hífen (-)

Exemplo:

descarte-esgoto-prainha.html
descarte-esgoto-prainha.jpg

💾 LIMITES DO GITHUB

  • Arquivo individual: até 100 MB
  • Recomendado por arquivo: até 25 MB
  • Repositório total: ideal até 1 GB

Importante:

  • Vídeos grandes → evitar
  • Preferir vídeos curtos ou compactados
  • Ou usar link externo (YouTube não listado)
